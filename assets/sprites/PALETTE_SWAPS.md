# Alien Sprite Palette Swaps

All alien sprites share the same saucer geometry (frog pilot, two-eye dome, elliptical disc with six rim lights, directional engine glow). Only the colour palette changes per type. Sprites are generated by `scripts/generate-alien-enemy-sprites.js`.

There are **8 directional variants** per type (right, diag-right-down, down, diag-left-down, left, diag-left-up, up, diag-right-up) ‚Äî 32 SVGs total for the enemy types below, plus 4 for the base frog (`alien-frog-*` from `generate-alien-saucer-sprites.js`).
